Andy Auth Documentation

Self-hosted OAuth 2.0 / OpenID Connect server built with ASP.NET Core, designed for seamless AI assistant integration via the Model Context Protocol (MCP).

Welcome to Andy Auth

Andy Auth provides a complete OAuth 2.0 and OpenID Connect implementation with native support for MCP OAuth 2.1 flows, enabling secure authentication for AI assistants like Claude Desktop, ChatGPT, and VS Code extensions.

Key Features

MCP Integration
Native OAuth 2.1 support for AI assistants including Claude Desktop, ChatGPT, and more.
OAuth 2.0 & OIDC
Standards-compliant implementation with Authorization Code, Client Credentials, and PKCE.
Dynamic Client Registration
RFC 7591/7592 compliant DCR for automated client onboarding.
Admin Dashboard
Complete UI for managing users, clients, tokens, and audit logs.

Quick Start

Start PostgreSQL
bash 
docker-compose up -d
Run the Server
bash 
cd src/Andy.Auth.Server
dotnet run
Access the Server

Server runs at https://localhost:7088

  • Email: test@andy.local

OAuth Endpoints

Endpoint URL Description
Authorization /connect/authorize Start OAuth flow
Token /connect/token Exchange code for tokens
UserInfo /connect/userinfo Get user claims
Introspection /connect/introspect Validate tokens
Revocation /connect/revoke Revoke tokens
Registration /connect/register Dynamic client registration
Discovery /.well-known/openid-configuration OpenID configuration
JWKS /.well-known/jwks JSON Web Key Set

MCP Support

Andy Auth provides native support for the Model Context Protocol (MCP) OAuth 2.1 specification, enabling secure authentication for AI assistants:

Claude Desktop
Configure Claude Desktop to authenticate with your MCP servers.
ChatGPT
Set up ChatGPT custom actions with OAuth authentication.
VS Code Extensions
Integrate Cline, Roo, and Continue.dev with OAuth.

Client Library

The Andy.Auth NuGet package provides easy integration for ASP.NET Core APIs:

bash 
dotnet add package Andy.Auth
csharp 
// Program.cs
builder.Services.AddAndyAuth(builder.Configuration);

Quick Start

Get Andy Auth running locally in under 5 minutes. This guide covers the fastest path to a working OAuth 2.1 authorization server.

Prerequisites
  • .NET 8 SDK
  • Docker Desktop (for PostgreSQL)
  • Git

Step 1: Clone the Repository

bash 
git clone https://github.com/rivoli-ai/andy-auth.git
cd andy-auth

Step 2: Start PostgreSQL

Start the database using Docker Compose:

bash 
docker-compose up -d

This starts PostgreSQL on port 5432 with:

  • Database: andy_auth_dev
  • Username: postgres
  • Password: postgres

Step 3: Run the Server

bash 
cd src/Andy.Auth.Server
dotnet run

The server will:

  1. Apply database migrations automatically
  2. Seed default OAuth clients for AI assistants
  3. Start listening on https://localhost:7088
Server Running!

You should see: Now listening on: https://localhost:7088

Step 4: Verify Installation

Open your browser and navigate to:

URL Description
https://localhost:7088 Home page
/.well-known/openid-configuration OpenID Discovery
/docs/ Documentation
/admin Admin Dashboard

Step 5: Create Your First User

Register a new account:

  1. Go to https://localhost:7088/Account/Register
  2. Enter your email and password
  3. Click Register

To make yourself an admin, run this SQL:

sql 
-- Connect to andy_auth_dev database
UPDATE "AspNetUsers"
SET "IsAdmin" = true
WHERE "Email" = 'your-email@example.com';

Test OAuth Flow

Try a quick authorization code flow using curl:

bash 
# 1. Open this URL in your browser (it will redirect to login)
open "https://localhost:7088/connect/authorize?\
client_id=claude-desktop&\
response_type=code&\
redirect_uri=http://localhost:3000/callback&\
scope=openid%20profile%20email&\
code_challenge=E9Melhoa2OwvFrEMTJguCHaoeK1t8URWbuGJSstw-cM&\
code_challenge_method=S256&\
state=abc123"

# 2. After login, you'll be redirected with an authorization code
# Example: http://localhost:3000/callback?code=ABC123&state=abc123

# 3. Exchange the code for tokens (use the code from step 2)
curl -k -X POST https://localhost:7088/connect/token \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=authorization_code" \
  -d "client_id=claude-desktop" \
  -d "code=YOUR_CODE_HERE" \
  -d "redirect_uri=http://localhost:3000/callback" \
  -d "code_verifier=dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk"
